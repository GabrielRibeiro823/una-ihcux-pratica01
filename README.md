# Meus Comandos Favoritos (CMD)

Aqui estão alguns comandos que utilizei durante a aula de Terminal no Windows:

## Navegação e Arquivos

* `cd` → Navegar entre pastas (Change Directory)
* `cd ..` → Voltar para a pasta anterior
* `dir` → Listar arquivos e pastas do diretório atual

## Outros Comandos Úteis

* `mkdir` → Criar uma nova pasta (Make Directory)
* `code .` → Abrir a pasta atual no Visual Studio Code
